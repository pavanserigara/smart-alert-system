# Directory Structure

```text

├── energy.php              # Energy Expert Dashboard

├── index.php               # Landing Page
```

## Key Files Summary
- **admin.php**: High-level management and analytics.
- **weather_engine.py**: The "Brain" of the ecosystem.
- **main.css**: The "Skin" of the ecosystem (Sci-Fi/Glassmorphic).
- **db_connect.php**: The "Nervous System" connecting logic to data.
